# pygame-snake

Jeu Snake codé en python à l'aide pygame.

## Comment jouer?

Pour démarer il faut appuyer sur `ESPACE` (peut-être un jour il y aura un menu).
Ensuite pour ce déplacer, utiliser les flèches.
